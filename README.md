# איקס-עיגול (Tic-Tac-Toe)

משחק איקס-עיגול קלאסי שפותח ב-HTML, CSS ו-JavaScript.

## שחק עכשיו
ניתן לשחק במשחק ישירות דרך הדפדפן בקישור הבא:
[לשחק עכשיו](https://liav-melamud.github.io/tic-tac-tok/)

## תכונות
* משחק איקס עיגול קלאסי ל-2 שחקנים (מקומי)
* עיצוב רספונסיבי מותאם למחשב ולטלפון נייד
* חיווי ויזואלי לתור נוכחי ולניצחון או תיקו

## הפעלה מקומית
ניתן פשוט להוריד את הפרוייקט ולפתוח את הקובץ `index.html` בתוך כל דפדפן. לא נדרשת התקנה מיוחדת או שרת.
